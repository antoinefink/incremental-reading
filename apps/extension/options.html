Interleave Browser capture

Pair the extension

Connect Chrome to the local Interleave desktop app. Captures stay on this machine through the token-protected loopback server.

Not connected Paste the token from Settings to pair this extension.

Pairing steps

  1. 1Open Interleave desktop.
  2. 2Go to Settings / Browser capture and copy the token.
  3. 3Paste it below, then test the connection.

Connection

Leave as-is unless Settings shows a different port.

Local-only. Captures travel only to http://127.0.0.1 on this machine. The desktop app validates the token and this extension's origin on every request.
